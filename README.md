# Ejecutar en Desarrollo

1. Clonar el repositorio
2. Instalar los paquetes npm

```
npm i --save
```

3. Tenes Nest CLI Instalado

```
npm i -g @nest/cli
```

4. Instalar Docker (omitir si ya se encuentra instalado)
5. Levantar el contenedor Docker

```
docker compose up -d
```

## Stack Usado

- MongoDB
- Nest
- TypeScript
- Moongose
